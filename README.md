### 51-SMP130
SMP130针式打印机头51单片机驱动程序<br /><br />

## 📥如何使用
void printerStitch(unsigned char str)   直接控制上部8针<br />
void printChar(unsigned char chr)       打印一个字符（自己画的字体，包括换行，@替换为黑方块）<br />
printStr(char *str)                     打印一个字符串<br />
